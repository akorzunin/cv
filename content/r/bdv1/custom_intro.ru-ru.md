# Backend разработчик

У меня **4 года** опыта коммерческой разработки. Владею языками **Python, Golang** для написания **REST** API сервисов, работы с базами данных **Postgres**, Redis и фреймворками FastAPI, Pydantic. Имею опыт в интеграции сервисов через REST и SOAP, разработке распределенных систем для обработки большого количества запросов с системой очередей на RabbitMQ. Есть опыт в создании автодокументации и web интерфейсов на **React** с использованием Tailwind.
